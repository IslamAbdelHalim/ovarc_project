## to run the application
- create a .env file in the root directory and add DB_HOST, DB_USER, DB_PASSWORD, DB_NAME, DB_PORT, SERVER_HOST, SERVER_PORT
and i am using mysql as relational database 

and you can use this .env for example

DB_HOST=mysql
DB_USER=root
DB_PASSWORD=passwrod
DB_NAME=ovarc
DB_PORT=3306

- and to run docker run this command =>  docker-compose up --build
